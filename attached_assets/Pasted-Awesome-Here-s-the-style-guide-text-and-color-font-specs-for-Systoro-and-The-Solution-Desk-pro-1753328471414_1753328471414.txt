# The Solution Desk & Systoro — Brand Style Guide (Draft)

---

## 1. Color Palette

| Color Name        | Hex     | Usage                         |
| ----------------- | ------- | ----------------------------- |
| **Deep Navy**     | #1D3557 | Headers, primary buttons      |
| **Warm Orange**   | #F4A261 | Highlights, secondary buttons |
| **Cream White**   | #FDF6E3 | Background                    |
| **Charcoal Gray** | #333333 | Body text                     |
| **Soft Blue**     | #A8DADC | Accent elements               |

---

## 2. Typography

### Headings (Warm, Professional Serif)

* **Primary Font:** *Lora*
* **Fallback:** Georgia, serif
* **Usage:** Page titles, section headers, main headings

### Body Text (Clean, Readable Sans-Serif)

* **Primary Font:** *Open Sans* or *Roboto*
* **Fallback:** Arial, sans-serif
* **Usage:** Paragraph text, tool descriptions, buttons, UI labels

---

## 3. Logo Usage

* Use Deep Navy or Cream White background for logo placement
* Maintain clear space around logo equal to the height of the logo icon
* Avoid stretching or altering proportions
* Use warm orange for highlights or accent parts if needed

---

## 4. UI Elements

### Buttons

* **Primary Buttons:** Deep Navy background, Cream White text, rounded corners (6–8px radius)
* **Secondary Buttons:** Warm Orange background, Cream White text
* **Hover State:** Slightly darker shade of button color

### Cards

* White or Cream White background
* Soft shadow for depth (e.g., `0 2px 6px rgba(0,0,0,0.1)`)
* Rounded corners consistent with buttons

---

## 5. Icon Style

* Simple line icons with rounded corners or soft edges
* Use Deep Navy or Soft Blue for main icon strokes
* Keep icons minimal and consistent in stroke weight
